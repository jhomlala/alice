# How to install

1. Add this to your **pubspec.yaml** file:

```yaml
dependencies:
  alice: ^1.0.0-dev8
```

2. Choose adapter based on your HTTP client. **pubspec.yaml** file:

### Dio
```yaml
dependencies:
  alice_dio: ^1.0.3
```

### Chopper
```yaml
dependencies:
  alice_chopper: ^1.0.3
```

### Http
```yaml
dependencies:
  alice_http: ^1.0.3
```

### Http Client
```yaml
dependencies:
  alice_http_client: ^1.0.3
```

3. Run `get` command:
```bash
$ flutter packages get
```
